OpenSSL

  • openssl_sign: gera uma assinatura digital a partir de uma chave privada
  • openssl_verify: verifica uma assinatura digital a partir de uma chave pública
  • Algoritmos disponíveis: http://php.net/manual/en/openssl.signature-algos.php ou openssl_get_md_methods()
